## 项目概述

小概率是一个独立开发者匹配平台，帮助项目方寻找技术合伙人，帮助开发者找到感兴趣的项目加入。

## 技术栈

| 端 | 技术 |
|---|------|
| Web 前台 | Nuxt 3 |
| 移动端 App | Flutter |
| 管理后台 | Vue 3 + Vite |
| 后端/数据库 | Supabase (PostgreSQL) |
| 部署 | Vercel |

## 核心信息

- **域名**: mirauni.com（已备案）
- **目标用户**: 独立开发者、创业者
- **商业模式**: 付费解锁联系方式（30元/10次）
