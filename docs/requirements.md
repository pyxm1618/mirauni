# 小概率独立开发者匹配平台 - 需求文档

## 1. 项目概述

### 1.1 产品定位
小概率是**独立开发者找合伙人的第一站**，帮助项目方发布项目寻找技术合伙人，帮助开发者找到感兴趣的项目加入。

### 1.2 目标用户
| 用户类型 | 描述 | 核心需求 |
|----------|------|----------|
| 项目方 | 有项目/想法的创业者、独立开发者 | 寻找技术合伙人、开发者 |
| 开发者 | 有技能、想参与项目的开发者 | 找到靠谱的项目加入 |

### 1.3 商业模式
- **核心收费点**：付费解锁联系方式（解锁后可站内信沟通）
- **付费方式**：次数包制
  - 基础包：30元/10次
  - 标准包：50元/30次
  - 高级包：100元/100次
- **辅助收入**：项目置顶、推荐位（二期）

## 6. 转化策略

### 6.1 浏览→注册
- 完全免费浏览项目和开发者
- 浏览3个详情页后弹出注册提示
- 详情页部分内容遮罩，提示注册后可见更多

### 6.2 注册→付费
- 想联系某人时，弹出付费提示
- 清晰展示价格和权益
- 首充优惠（限时+加送）

### 6.3 首充优惠
- 新用户首充+30%次数
- 注册后24小时内首充8折
- （具体数值可调整）

### 7.1 内容种子
- 发布自有项目（3-5个）

- V2EX发帖

### 7.3 SEO策略
- Nuxt 3 SSR保证内容可被抓取
- 提交百度sitemap
- 主动推送API
- 学院内容长尾关键词

## 9. 移动端规划

### 9.1 技术选型
- **框架**：Flutter
- **平台**：iOS + Android
- **发布渠道**：App Store、应用宝、华为、小米等

### 9.2 功能范围

- 项目列表/详情浏览

### 9.3 专属功能
- 消息推送通知
